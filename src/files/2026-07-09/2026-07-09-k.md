### Why the Engine Failed

Right now, in **Phase 2 (Standard Trait Counting)**, the engine only looks at the *immediate* array of the word.

Let's look at the `golden retriever` and `pitbull` arrays:

* `golden retriever`: `["dog", "friendly", "water", "fetch", "gentle"]`
* `pitbull`: `["dog", "strong", "guard", "danger", "muscular", "teeth"]`

The engine asked: *"What words do these two lists share?"* The answer is only **ONE** word: `"dog"`. So it scored them a miserable 20%.

The engine forgot that because they are both dogs, they *also* share "fur", "bark", "tail", "canine", "mammal", "warm-blooded", and "DNA"! But those words are hidden higher up the ancestry tree. The engine only looked at the tip of the iceberg.

Similarly, `bird` has `"wings"`, but `airplane` only has `"aircraft"`. The `"wings"` trait is hidden inside the `aircraft` parent array.

### The Solution: Trait Compilation (Flattening)

Before the engine compares two words, it must **compile** them. It needs to run up the ancestry tree, grab every single trait from every single parent, and dump them into one massive "Master Array" for each word.

// --- NEW HELPER FUNCTION: Gather all inherited traits ---
function gatherAllTraits(word)
{
    let allTraits = [];
    let current = word;
    let limit = 20; // Safety limit
    let steps = 0;

    while (cortex[current] !== undefined && steps < limit)
    {
        // 1. Dump all traits from the current level into our master list
        for (let i = 0; i < cortex[current].length; i++)
        {
            let trait = cortex[current][i];
            // Only add it if we don't already have it!
            if (!allTraits.includes(trait)) 
            {
                allTraits.push(trait);
            }
        }

        // 2. Find the parent to climb up the tree
        let parent = cortex[current][0];
        
        // 3. Stop if we hit the root or a loop
        if (parent === "root" || parent === current) 
        {
            break; 
        }
        
        // 4. Move up to the next level
        current = parent;
        steps++;
    }
    
    return allTraits;
}
